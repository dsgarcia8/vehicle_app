# Vehicle app

## Env template

Create a `.env` file with as the following template

```
DB_HOST=xxxxxx
DB_USER=xxxxxx
DB_PASSWORD=xxxxxx
DB_NAME=sxxxxx
```

## Run project

  Install dependencies:

```console
npm install
```

  Start the server:

```console
npm start
```

  View the website at: <http://localhost:3000>
